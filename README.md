Trabalho de Atividade Prática Supervisionada referente ao 8° semestre do curso de Ciencia da Computação da Universidade Paulista, na qual foi criado um APP de Android utilizando APIs para a busca da temperatura e da qualidade do ar da cidade pesquisada.

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
